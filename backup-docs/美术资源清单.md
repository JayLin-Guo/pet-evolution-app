# 美术资源清单

## 资源命名规范

### 序列帧动画命名
```
pet_{stage}_{substage}_{animation}_{frame}.png

例如：
pet_baby_1_idle_00.png
pet_baby_1_idle_01.png
...
pet_child_2_happy_00.png
```

### 部件资源命名
```
pet_part_{partname}_{variant}.png

例如：
pet_part_claws_small.png
pet_part_claws_large.png
pet_part_scales_01.png
pet_part_wings_small.png
```

## 需要的资源列表

### 一、基础动画（每个阶段都需要）

#### 1. 幼兽期（baby）
- 阶段 1-3，每个阶段需要：
  - idle（待机）：8帧
  - happy（开心）：12帧
  - eat（吃东西）：10帧
  - sleep（睡觉）：6帧
  - play（玩耍）：16帧

**小计：** 3个小阶段 × 5个动画 × 平均10帧 = 150张图

#### 2. 成长期（child）
- 阶段 1-3，动画同上
- **新增部件：** 爪子（3个变体）

**小计：** 150张图 + 3张部件

#### 3. 青年期（teen）
- 阶段 1-3，动画同上
- **新增部件：** 鳞片（5个变体）、小翅膀（3个变体）

**小计：** 150张图 + 8张部件

#### 4. 成年期（adult）
- 阶段 1-3，动画同上
- **新增部件：** 完整翅膀（3个变体）、更多鳞片（5个变体）

**小计：** 150张图 + 8张部件

#### 5. 壮年期（prime）
- 阶段 1-3，动画同上
- **新增部件：** 光环（3个变体）、特效（5个变体）

**小计：** 150张图 + 8张部件

#### 6. 巅峰期（peak）
- 阶段 1-3，动画同上
- **新增部件：** 终极形态特征（每种形态5个变体）

**小计：** 150张图 + 25张部件（5种形态）

### 二、特殊动画

#### 进化动画（evolve）
- 每个阶段过渡：20帧
- 共需要：5个过渡 × 20帧 = 100张图

#### 升级特效
- 闪光效果：10帧
- 可复用

### 三、UI 元素

#### 图标
- 喂食按钮图标
- 玩耍按钮图标
- 抚摸按钮图标
- 聊天按钮图标

#### 背景
- 主界面背景
- 不同时间段的背景变化（早中晚）

## 制作建议

### 方案一：像素风格（推荐新手）
**工具：** Aseprite, Piskel
**优势：**
- 制作快速
- 文件小
- 风格统一容易
- 适合独立开发

**尺寸建议：** 64×64 或 128×128 像素

### 方案二：卡通风格
**工具：** Procreate, Clip Studio Paint
**优势：**
- 表现力强
- 可爱度高
- 适合大众市场

**尺寸建议：** 512×512 或 1024×1024 像素

### 方案三：3D渲染
**工具：** Blender, Maya
**优势：**
- 可以自由旋转
- 光影效果好
- 后期调整方便

**尺寸建议：** 1024×1024 像素

## 资源优化

### 文件格式
- PNG（透明背景）
- WebP（更小体积）
- 压缩工具：TinyPNG

### 图集打包
- 使用 TexturePacker 打包成图集
- 减少文件数量
- 提高加载性能
