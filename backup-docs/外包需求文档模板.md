# 虚拟宠物动画外包需求文档

## 项目概述
虚拟宠物养成 App，需要制作宠物在不同成长阶段的动画素材。

## 美术风格
**待定，请提供以下风格的报价：**
- [ ] 像素风格（推荐，参考：Tamagotchi、宝可梦）
- [ ] 卡通风格（参考：摩尔庄园、洛克王国）
- [ ] Q版风格（参考：阴阳师式神）

## 需求清单

### 一、MVP 最小版本（优先）

#### 需要的动画：
1. **幼兽期 - 阶段1**
   - 待机动画：8帧，循环播放
   - 开心动画：12帧，播放一次
   - 吃东西动画：10帧，播放一次
   
2. **成长期 - 阶段1**
   - 待机动画：8帧
   - 开心动画：12帧
   - 吃东西动画：10帧
   - **新增特征：** 爪子变大

3. **青年期 - 阶段1**
   - 待机动画：8帧
   - 开心动画：12帧
   - 吃东西动画：10帧
   - **新增特征：** 身上有鳞片

**小计：** 3个阶段 × 3个动画 × 平均10帧 = 约90张图

### 二、完整版本（后期）

#### 6个大阶段，每个阶段3个小阶段
1. 幼兽期（baby）- 像小猫
2. 成长期（child）- 长出爪子
3. 青年期（teen）- 长出鳞片和小翅膀
4. 成年期（adult）- 翅膀变大
5. 壮年期（prime）- 有光环特效
6. 巅峰期（peak）- 终极形态

#### 每个阶段需要的动画：
- 待机（idle）：8帧
- 开心（happy）：12帧
- 吃东西（eat）：10帧
- 睡觉（sleep）：6帧
- 玩耍（play）：16帧

**小计：** 约900张图

### 三、终极形态（5种）
根据属性不同，最终进化为：
1. 神龙（威武，有龙角龙鳞）
2. 饕餮（凶猛，大嘴巴）
3. 天使兽（圣洁，有光环翅膀）
4. 凤凰（优雅，火焰翅膀）
5. 麒麟（祥瑞，有角有鳞）

每种形态需要：待机动画8帧

**小计：** 5种 × 8帧 = 40张图

## 技术要求

### 文件格式
- PNG 格式（透明背景）
- 分辨率：512×512 像素（或根据风格调整）
- 每帧独立文件

### 命名规范
```
pet_{阶段}_{子阶段}_{动画}_{帧数}.png

例如：
pet_baby_1_idle_00.png
pet_baby_1_idle_01.png
pet_child_1_happy_00.png
```

### 交付内容
1. 所有动画序列帧（PNG）
2. 源文件（PSD/AI/Aseprite等）
3. 动画预览 GIF
4. 简单的使用说明

## 谈判技巧

### 如何降低成本
1. **先做 MVP**（90张），测试效果
2. **分批制作**，不要一次全做
3. **找新人画师**，价格更便宜
4. **提供详细参考**，减少修改次数
5. **长期合作**，可以谈折扣

### 如何判断报价是否合理
- 像素风：15-30元/张
- 卡通风：30-60元/张
- 精美风：60-100元/张

### 红线（不要接受）
- 不提供源文件
- 不允许修改
- 价格过高（超过上述2倍）
- 交付时间过长（MVP超过1个月）
