# 快速开始指南

## 第一步：检查环境

打开命令行，运行：
```bash
node --version
npm --version
```

如果显示版本号，说明已安装。如果没有，请先安装 Node.js：
https://nodejs.org/

## 第二步：选择方案

### 推荐：Expo（最简单）

**优势：**
- ✅ 不需要 Android Studio 或 Xcode
- ✅ 扫码就能在手机上运行
- ✅ 开发速度最快
- ✅ 热重载超快

**创建项目：**
```bash
npx create-expo-app@latest . --template blank-typescript
```

### 备选：原生 React Native

**优势：**
- 性能更好
- 更灵活

**劣势：**
- 需要配置 Android Studio
- 首次配置较复杂

**创建项目：**
```bash
npx react-native@latest init PetApp --template react-native-template-typescript
```

## 第三步：运行项目

### 如果用 Expo：

1. **启动开发服务器**
```bash
npx expo start
```

2. **在手机上查看**
   - 下载 Expo Go App（iOS App Store 或 Android Play Store）
   - 扫描终端显示的二维码
   - 立即看到效果！

3. **或在模拟器查看**
   - 按 `a` 打开 Android 模拟器
   - 按 `i` 打开 iOS 模拟器（需要 Mac）

### 如果用原生 RN：

```bash
# Android
npx react-native run-android

# iOS (需要 Mac)
npx react-native run-ios
```

## 常见问题

### Q: 我应该选 Expo 还是原生 RN？
**A:** 选 Expo！除非你有特殊需求，否则 Expo 更简单快速。

### Q: 需要安装 Android Studio 吗？
**A:** 如果用 Expo，不需要！如果用原生 RN，需要。

### Q: 可以在 Windows 上开发 iOS 应用吗？
**A:** 用 Expo 可以！扫码在 iPhone 上查看。但打包 iOS 应用需要 Mac。

### Q: 项目创建失败怎么办？
**A:** 
1. 确保 Node.js 版本 >= 18
2. 清除 npm 缓存：`npm cache clean --force`
3. 重试
